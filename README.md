# CQLToFilterTree
CQL To Filter Tree XML Conversion Utility

### Run Instructions
`mvn clean compile assembly:single`

Run the jar from the target dir `java -jar JAR_NAME`

Input a CQL string like `attrName >= 5` and the XML will log to the terminal.
